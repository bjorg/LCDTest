Installation
------------
To install LCDTest, drag the LCDTest icon to your Application folder.

Uninstallation
--------------
To uninstall LCDTest, drag the LCDTest icon from your Application folder to the Trash icon.

Building
--------
The source code can be built with XCode v1.1 on OS X 10.3 (Panther).
